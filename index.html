HTML PHP y MySQL

HTML: Es como la estructura de una página web. Define cómo se organizan los elementos (títulos, párrafos, imágenes).
PHP: Es el cerebro detrás de la página. Permite crear contenido dinámico, interactuar con bases de datos y realizar acciones más complejas.

HTML

HTML (HyperText Markup Language) es el lenguaje estándar que se utiliza para crear la estructura de las páginas web. Imagina que es como el esqueleto de una casa: define dónde van los encabezados, los párrafos, las imágenes, los enlaces y todos los demás elementos que ves en una página web

MySQL

es un sistema de gestión de bases de datos relacionales muy popular. Imagina una base de datos como una biblioteca gigante, pero en lugar de libros, almacena datos organizados en tablas. MySQL es el bibliotecario de esta biblioteca, encargado de ordenar, buscar y gestionar toda la información.

PHP

Se utiliza principalmente para crear el backend de las páginas web, es decir, la parte que los usuarios no ven directamente pero que hace que las páginas sean dinámicas e interactivas.
