¿Qué es HTML?

HTML (HyperText Markup Language) es el lenguaje de marcado estándar utilizado para crear páginas web. Permite estructurar el contenido de una página mediante etiquetas y atributos.

Beneficios de HTML

  • Es fácil de aprender.
  • Se usa para estructurar contenido web.
  • Es compatible con todos los navegadores web.

¿Dónde se utilizan las páginas web?

Las páginas web se utilizan en una gran variedad de aplicaciones, desde simples blogs hasta portales complejos de comercio electrónico.

Diferencia entre páginas web estáticas y dinámicas

Una página estática tiene un contenido fijo que no cambia. En cambio, una página dinámica se actualiza según las interacciones del usuario o los datos de un servidor.

¿Qué es un portal web?

Un portal web es un sitio web que sirve como punto de acceso a diferentes servicios, herramientas o información dentro de una organización.

1. Encabezados

Los encabezados se utilizan para dividir y organizar el contenido de una página. Se usan etiquetas como <h1>, <h2>, <h3>....

Ejemplo de uso de encabezados:

<h1>Este es un encabezado h1</h1>

Ejercicio:

Crea un encabezado de tipo h1 para el título de tu página, y un h2 para un subtítulo.

2. Párrafos

Los párrafos son elementos fundamentales para organizar texto en una página web. Se definen con la etiqueta <p>.

Ejemplo de uso de párrafos:

<p>Este es un párrafo.</p>

3. Formateo de Texto

HTML ofrece etiquetas como <strong> y <em> para resaltar texto. Texto en negrita y texto en cursiva.

4. Fuentes

Las fuentes se definen con CSS o con las etiquetas de texto de HTML. Puedes cambiar la familia, tamaño y color de la fuente.

PHP: Introducción

PHP es un lenguaje de programación utilizado para crear sitios web dinámicos. Es ampliamente usado en el desarrollo de aplicaciones web.

¿Qué es PHP y para qué se utiliza?

PHP es un lenguaje de programación de código abierto que se usa para crear aplicaciones web dinámicas, generando contenido en tiempo real.

Instalación y requisitos para PHP

Para usar PHP, necesitas tener instalado un servidor web, como Apache, y un interprete PHP.
