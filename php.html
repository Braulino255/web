Guía para Aprender PHP y MySQL

En este manual aprenderás lo básico sobre PHP, MySQL y cómo trabajar con bases de datos en la web.

1. ¿Qué es PHP y para qué se utiliza?

PHP es un lenguaje de programación de código abierto que se utiliza para el desarrollo de aplicaciones web dinámicas. PHP se ejecuta en el servidor y puede generar contenido HTML de manera dinámica.

Se utiliza principalmente para crear formularios, gestionar bases de datos y generar contenido dinámico en las páginas web.

2. ¿Qué programas requiero para poder utilizar PHP?

Para usar PHP necesitas instalar un servidor web como Apache o Nginx y un servidor de base de datos como MySQL o MariaDB. Además, se necesita instalar PHP.

En sistemas locales, puedes usar un paquete como XAMPP o WAMP, que incluyen Apache, MySQL y PHP en un solo instalador.

3. ¿Qué es un servidor web?

Un servidor web es un software que se encarga de recibir las solicitudes de los usuarios a través de Internet y servirles páginas web. Algunos ejemplos son Apache, Nginx y Microsoft IIS.

4. ¿Qué es MySQL y qué diferencia hay entre SQL y MySQL?

MySQL es un sistema de gestión de bases de datos relacional de código abierto. SQL (Structured Query Language) es el lenguaje estándar utilizado para interactuar con bases de datos, y MySQL es un software que implementa SQL para gestionar bases de datos.

En resumen, SQL es el lenguaje y MySQL es el software que lo utiliza para gestionar la información.

5. ¿Cómo se usa PHP directamente en una página web?

Para utilizar PHP en una página web, simplemente debes incluir código PHP dentro de un archivo con extensión .php. El código PHP se coloca entre las etiquetas <?php ?>.

<?php
    echo "¡Hola Mundo!";
?>

6. ¿Qué son las variables y para qué sirven?

Las variables en PHP son espacios de almacenamiento que pueden contener valores como números, texto o listas de valores. Se crean usando el signo de dólar $, seguido del nombre de la variable.

$nombre = "Juan"; // Variable de tipo cadena

Las variables se usan para almacenar datos que luego pueden ser manipulados dentro del programa.

7. ¿Cuál es la variable string y qué función tiene?

La variable string en PHP se usa para almacenar texto. Puedes crear una variable de tipo string poniendo el texto entre comillas.

$saludo = "Hola, ¿cómo estás?";

Las funciones de string permiten realizar operaciones con texto, como concatenar, buscar o reemplazar partes de cadenas.

8. ¿Cómo se usa la estructura if en PHP y para qué sirve?

La estructura if en PHP se utiliza para ejecutar un bloque de código solo si se cumple una condición.

if ($edad >= 18) {
    echo "Eres mayor de edad";
} else {
    echo "Eres menor de edad";
}

En este ejemplo, se evalúa si la variable $edad es mayor o igual a 18 para determinar si la persona es mayor de edad.

9. ¿Cuáles son los repetidores que se usan en PHP y cómo se ocupan?

PHP tiene varios tipos de estructuras para repetir bloques de código, entre ellas están for, foreach, while y do-while.

// Ejemplo de foreach
$frutas = array("Manzana", "Banana", "Cereza");
foreach ($frutas as $fruta) {
    echo $fruta . "
";
}

En este ejemplo, foreach recorre cada elemento del array $frutas y lo imprime.

10. ¿Cómo crear un formulario?

Para crear un formulario en PHP, se utiliza HTML con el atributo action para indicar a qué archivo se enviarán los datos.

<form action="procesar.php" method="POST">
    <input type="text" name="nombre" placeholder="Escribe tu nombre">
    <input type="submit" value="Enviar">
</form>

11. ¿Cómo crear un formulario y guardar la información en un bloc de notas?

PHP puede guardar la información de un formulario en un archivo de texto (bloc de notas) usando la función file_put_contents().

<?php
if ($_SERVER["REQUEST_METHOD"] == "POST") {
    $nombre = $_POST["nombre"];
    file_put_contents("datos.txt", $nombre . "\n", FILE_APPEND);
}
?>

12. ¿Cómo crear una base de datos en MySQL?

Para crear una base de datos en MySQL, puedes usar el siguiente comando SQL:

CREATE DATABASE nombre_base_datos;

13. ¿Cómo crear una tabla en MySQL?

Para crear una tabla, usas el comando SQL CREATE TABLE con las columnas que desees.

CREATE TABLE usuarios (
    id INT AUTO_INCREMENT PRIMARY KEY,
    nombre VARCHAR(100),
    correo VARCHAR(100)
);

14. ¿Cómo crear un registro en MySQL?

Para insertar datos en una tabla, usas el comando SQL INSERT INTO.

INSERT INTO usuarios (nombre, correo) VALUES ("Juan", "juan@ejemplo.com");

15. ¿Cómo almacenar información de un formulario en MySQL?

Puedes usar PHP para tomar los datos del formulario y almacenarlos en una base de datos MySQL con el siguiente código:

<?php
if ($_SERVER["REQUEST_METHOD"] == "POST") {
    $nombre = $_POST["nombre"];
    $conn = new mysqli("localhost", "usuario", "contraseña", "base_datos");
    $sql = "INSERT INTO usuarios (nombre) VALUES ('$nombre')";
    $conn->query($sql);
}
?>

16. ¿Cómo consultar información de una base de datos?

Para consultar información de una base de datos, puedes usar el comando SQL SELECT.

<?php
$nombre = $_GET["nombre"];
$conn = new mysqli("localhost", "usuario", "contraseña", "base_datos");
$sql = "SELECT * FROM usuarios WHERE nombre = '$nombre'";
$result = $conn->query($sql);
?>
