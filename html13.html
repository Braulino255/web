Guía para Aprender HTML

En esta guía aprenderás sobre los principales elementos HTML para crear páginas web.

1. Encabezados

Los encabezados son usados para definir títulos en una página web. HTML ofrece 6 niveles de encabezados, siendo <h1> el más grande y <h6> el más pequeño.

<h1>Este es un encabezado de nivel 1</h1>

2. Párrafos

Los párrafos en HTML se definen con la etiqueta <p>. Cada párrafo debe estar dentro de esta etiqueta para separarlos y organizar el contenido de manera adecuada.

<p>Este es un párrafo de ejemplo.</p>

Ejemplo de un párrafo:

3. Formateo de texto

HTML ofrece etiquetas para aplicar formato al texto, como <b> para negrita, <i> para cursiva y <u> para subrayado.

<b>Texto en negrita</b>

Ejemplo de formateo:

Texto en negrita, Texto en cursiva y Texto subrayado.

4. Fuentes

Para cambiar la fuente del texto en HTML, se puede usar la propiedad font-family dentro de CSS. Puedes usar fuentes predeterminadas o fuentes personalizadas que estén disponibles en tu servidor o mediante servicios como Google Fonts.

p {
    font-family: 'Arial', sans-serif;
}

5. Los tres tipos principales de listas en HTML

HTML tiene tres tipos principales de listas: numeradas, sin numerar y glosarios.

5.1. Listas numeradas

Las listas numeradas se definen con <ol>, y cada ítem de la lista se coloca dentro de una etiqueta <li>.

<ol>
    <li>Primer elemento</li>
    <li>Segundo elemento</li>
</ol>

Ejemplo de lista numerada:

  1. Primer elemento
  2. Segundo elemento

5.2. Listas sin numerar

Las listas sin numerar se definen con <ul>.

<ul>
    <li>Elemento sin numerar</li>
</ul>

Ejemplo de lista sin numerar:

  • Elemento sin numerar
  • Otro elemento sin numerar

5.3. Listas de glosario

Las listas de glosario se crean con <dl> (definition list), <dt> para los términos y <dd> para las definiciones.

<dl>
    <dt>HTML</dt>
    <dd>Lenguaje de marcado para estructurar páginas web</dd>
</dl>

Ejemplo de lista de glosario:

HTML
Lenguaje de marcado para estructurar páginas web
CSS
Lenguaje para dar estilo a las páginas web

6. Manejo de Gráficas en HTML

6.1. Alineamiento de la gráfica en la página

Las imágenes se alinean en la página utilizando las propiedades de CSS como text-align o margin.

img {
    display: block;
    margin: 0 auto;
}

Ejemplo de alineación centrada:

6.2. Alineamiento de texto en relación a la gráfica

Se puede alinear el texto alrededor de la gráfica usando la propiedad float.

img {
    float: left;
    margin-right: 20px;
}

Ejemplo de texto alrededor de una imagen:

Este texto está alineado al lado de la imagen que se encuentra a la izquierda.

6.3. Espacios alrededor de la gráfica y tamaño

Para controlar los márgenes alrededor de la gráfica, puedes usar margin y padding.

img {
    margin: 20px;
    width: 100px;
}

6.4. Gráfica de fondo para una página

Para poner una imagen como fondo de una página, se utiliza CSS en el selector body.

body {
    background-image: url('fondo.jpg');
    background-size: cover;
}
